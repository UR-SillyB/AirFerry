## v1.1.0

### 性能

- **发送端 fixed-mask QR 编码**：会话内帧长恒定，vendor/fast_qr 在指定 mask 时跳过 8-mask 评估（约再提速一个数量级于已换用的 fast_qr Reed-Solomon 路径）。热路径仍为 `next_qr_*_into` + JS `drawMatrix`（WASM 直出 RGBA 因跨边界拷贝更重，未采用）。
- **发送档位**：新增 turbo（1900B）/ max（2400B）符号预设；默认仍为激进 1400B。
- **Android 分析流保持 1080p**：ImageAnalysis 请求 1920×1080，优先保证高版本/多码下的模块像素密度与识别率。

### 其他

- 清理未启用的 WASM RGBA API 与实验分支。
- 版本号四处同步至 1.1.0（`package.json` / `build.gradle.kts` versionCode=5 / `Cargo.toml` / `AirFerry.Windows.csproj`）。
